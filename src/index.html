Website Absensi Karyawan

Website untuk melakukan absensi/presensi kehadiran karywan, mengelola data pegawai dan absensi. Terdapat Admin yang mendaftarkan pegawai baru dan mengelola data pegawai. Terdapat pegawai yang melakukan absensi.

Teknologi yang digunakan adalah html, css, javascript, bootstrap 5, php, dan mysql.
